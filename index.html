Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web. El contenido es completado por el alumno.

Comenzar

¿Qué es HTML5?

HTML5 es el lenguaje de marcado estándar para crear y estructurar contenido en la web. Sirve para definir la estructura básica de una página, organizando elementos como texto, imágenes y enlaces, siendo fundamental para el desarrollo web moderno al proporcionar una base sólida y compatibilidad con múltiples dispositivos.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

- DOCTYPE: Declara el tipo de documento y la versión de HTML que se está utilizando. - html: Etiqueta raíz que engloba todo el contenido de la página. - head: Contiene información meta sobre la página, como el título, enlaces a estilos o scripts. - body: Incluye todo el contenido visible para el usuario, como texto, imágenes y elementos interactivos.

Etiquetas principales

h1 - h6

Las etiquetas de título definen niveles de encabezado, desde h1 (el más importante) hasta h6 (el menos relevante), ayudando a estructurar el contenido y mejorar la accesibilidad y el SEO.

p

La etiqueta de párrafo se utiliza para agrupar bloques de texto continuo, organizando la información de forma legible.

img

Permite insertar imágenes en la página. Requiere atributos como src (ruta de la imagen) y alt (texto alternativo para accesibilidad).

a

Crea enlaces a otras páginas web, archivos o secciones dentro de la misma página. El atributo href define la dirección de destino.

button

Genera botones interactivos que pueden desencadenar acciones cuando son clicados, como enviar formularios o ejecutar scripts.

ul / ol / li

ul crea listas desordenadas (con viñetas), ol listas ordenadas (con números o letras), y li define cada elemento de la lista.

HTML5 semántico

header

Representa el encabezado de una sección o de la página, generalmente contiene logotipos, títulos y navegación principal.

nav

Define una sección de navegación que contiene enlaces a otras páginas o secciones del sitio.

main

Encierra el contenido principal y único de la página, excluyendo elementos como encabezados, pies de página o barras laterales.

section

Agrupa contenido temáticamente relacionado, como capítulos, secciones de artículos o bloques de funcionalidades.

article

Representa un contenido independiente y autónomo que podría ser distribuido o reutilizado por sí mismo, como un artículo de blog o una noticia.

footer

Define el pie de una sección o de la página, habitualmente contiene información de autoría, derechos de autor o enlaces relacionados.

Formularios en HTML5

Los formularios en HTML5 permiten recopilar datos del usuario. Algunas etiquetas y elementos clave son: form (define el formulario), label (etiqueta descriptiva para campos), input (campo de entrada de datos, con tipos como text, email), textarea (campo de texto multilínea), select (lista desplegable) y button (botón para enviar o resetear el formulario).

Multimedia en HTML5

HTML5 permite integrar contenido multimedia directamente en la página sin necesidad de plugins adicionales, facilitando la inclusión de imágenes, audio y video para enriquecer la experiencia del usuario.

Imágenes

La etiqueta img se usa para insertar imágenes; sus atributos principales son src (ruta del archivo), alt (texto alternativo para accesibilidad y cuando la imagen no carga), width y height (dimensiones de la imagen).

Audio

La etiqueta audio permite reproducir archivos de audio. Admite formatos como MP3, WAV y OGG. Incluye atributos como controls (para mostrar controles de reproducción), src (ruta del archivo) y autoplay (para reproducir automáticamente, aunque su uso está limitado por navegadores).

Video

La etiqueta video inserta videos en la página, soportando formatos como MP4, WebM y OGG. Sus atributos incluyen controls, src, width, height, autoplay y poster (imagen que se muestra antes de reproducir el video).

Accesibilidad

El atributo alt en la etiqueta img es fundamental para la accesibilidad, ya que proporciona una descripción textual de la imagen a usuarios con lectores de pantalla o cuando la imagen no puede ser cargada, garantizando que todo el contenido sea comprensible para todos.

Conclusión

HTML5 es una herramienta esencial en el desarrollo web moderno, ya que no solo estructura el contenido de las páginas, sino que también mejora la accesibilidad, la compatibilidad entre dispositivos y la integración de elementos multimedia y semánticos. Su uso adecuado permite crear sitios web más organizados, eficientes y fáciles de mantener, siendo la base sobre la cual se construyen experiencias digitales de calidad.
